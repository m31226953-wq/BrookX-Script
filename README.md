
## 🎮 Функции

| Функция | Описание |
|---------|----------|
| **Noclip Mode+** | Проход сквозь стены (вкл/выкл) |
| **Invisible Mode+** | Полная невидимость персонажа |
| **Jump Mode** | Бесконечные прыжки |
| **Rainbow Tag** | Радужный никнейм над головой |
| **Copy Skin** | Копирование скина любого игрока |
| **Sound Player** | Проигрывание звука на сервере (3 сек) |
| **Player Follow** | Слежка за игроком |
| **Kill Mode** | Убийство игрока с помощью тележки |
| **Player Detector** | Показывает всех игроков на сервере |
